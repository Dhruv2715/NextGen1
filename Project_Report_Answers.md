### 1. Project Title
**NextGen - AI Technical Assessment Portal**

---

### 2. Project Description
A dual-portal (Interviewer & Candidate) platform for technical assessments featuring modern UI, real-time job tracking, and career discovery tools.

---

### 3. Project Goal
To simplify technical hiring for recruiters while providing candidates with better job discovery and practice tools on a single platform.

---

### 4. Objectives
- Build role-based access for Interviewers/Candidates.
- Enable real-time job posting and status management.
- Provide talent analytics and hiring funnel insights.

---

### 5. Functional Requirements
- Secure JWT Authentication.
- Job CRUD operations & status toggling.
- Real-time search/filtering for jobs and candidates.
- Candidate profile and application tracking.

---

### 6. Non-Functional Requirement
- Scalability to handle multiple users.
- Fast performance and role-based security.
- Modern Glassmorphism responsive design.

---

### 7. Hardware Specification
- Laptop/PC: i5 Processor, 8GB RAM, stable internet connection.

---

### 8. Software Specification
- MERN Stack: MongoDB, Express, React, Node.js.
- Styling: Tailwind CSS.
- Security: JWT (JSON Web Tokens).

---

### 9. Learning Outcome
- Proficiency in MERN stack development and secure authentication workflows.
- Experience in building responsive UI and state management in React.
